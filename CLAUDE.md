# Gaia - Marketing Agency Website

## Project Overview

**Name:** Gaia
**Industry:** Marketing Agency, Advertising & Web Development
**Project Type:** Informational Website + Portfolio

## Objective

Build a modern, professional, and feminine informational website with portfolio for a marketing agency that offers:

- Marketing digital
- Publicidad (Advertising)
- Branding
- Desarrollo web (Web Development)
- Redes sociales (Social Media)
- Contenido creativo (Creative Content)

## Visual Inspiration

**Reference:** holasomosmodo.com

### Design Principles

- Elegant
- Smooth animations
- Modern typography
- Professional and feminine aesthetic


## Brand Identity

### Color Palette

ALWAYS start by scanning the `global.css` and use the color variables from there.  
- Burgundy as primary color
- Pink as secondary color
- Beige as tertiary color

### Typography

The design should represent or approximate these brand typefaces:

- **Codec Pro**
- **Giaza**
- **Symphony**
- **Condec Pro**

### Brand Application Guidelines

- Maintain an elegant, subtle, and feminine aesthetic aligned with branding. inspired by contemporary premium aesthetics, boutique branding, and modern editorial fashion

## General Style

- Professional
- Clean
- Coherent with the color palette
- Elegant transitions

## Technical Stack

- Next.js (React framework)
- TypeScript
- Tailwind CSS for styling
- Modern web development best practices

## Design Direction

The website should feel:
- Sophisticated yet approachable
- Feminine without being overly decorative
- Smooth and refined in interactions
- Focused on showcasing work and services clearly
